### Modes

- playing
    - grid size depand on terminal size
    - 10% margin

- insert
    - game is paused when exiting insert mode

### Keybinds

##### playing
- [x] `q`: quit game
- [x] `+`: faster
- [x] `-`: slower
- [x] `p`: pause
- [x] `s`: step
- [x] `i`: inset mode

##### insert
- [x] `q` quit
- [x] `escape`: quit insert mode
- [x] `t`: toggle cell at cursor position
- [x] `arrows/hjkl`: move arround
- [x] `c`: clear
- [x] `r`: random generation

##### CLI
- [ ] fps
- [ ] paused

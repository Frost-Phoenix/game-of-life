### Modes

- playing

    -  10% margin

- insert
    - game is paused when exiting insert mode

### Keybinds

##### playing
- [ ] `escape/q`: quit game
- [ ] `+`: faster
- [ ] `-`: slower
- [ ] `p`: pause
- [ ] `s`: step
- [ ] `i`: inset mode

##### insert
- [ ] `escape/q`: quit insert mode
- [ ] `space`: toggle cell at cursor position
- [ ] `arrows/hjkl`: move arround
- [ ] `c`: clear
- [ ] `r`: random generation
